# 📝 CAN

> **"당신은 할 수 있습니다 (You CAN)."**  
시대를 초월한 명언과 사용자의 마음을 연결하고, 서로를 응원하는 모바일 전용 동기부여 플랫폼입니다.

---

## 📌 1. 프로젝트 개요 (Overview)
* **목적**: 모바일에 최적화된 단순하고 직관적인 UI를 통해 매일 긍정적인 자극을 전달하며, 맞춤형 명언 추천 및 소통 공간(커뮤니티)을 제공하여 사용자의 지속적인 동기부여를 돕습니다.
* **타겟 플랫폼**: Mobile Only (iOS / Android)
* **핵심 슬로건**: 일상에 작은 변화를 만드는 단 하나의 앱, CAN.

---

## 📱 2. 모바일 특화 UI/UX 및 하단 탭 구조

모바일 환경에서의 편안한 사용성을 위해 **한 손 조작(Thumb-Zone)**을 고려한 UI 구조와 불필요한 단계를 제거한 미니멀 디자인을 지향합니다. 모든 주요 기능은 화면 하단의 **5개 탭**을 통해 직관적으로 접근할 수 있습니다.

### 🎨 테마 색상 커스터마이징 (Theme Settings)
* **나만의 무드 설정**: 앱 전체의 포인트 컬러(예: 차분한 파스텔 블루, 따뜻한 인디핑크, 활기찬 옐로우 등)를 사용자가 선호하는 테마 색상으로 간편하게 변경할 수 있습니다.
* **다크 모드 완벽 지원**: 야간이나 이른 아침에도 눈 피로 없이 명언을 읽고 알람을 설정할 수 있도록 다크 테마를 기본 제공합니다.

### 🗺️ 하단 5개 탭 구조 (Bottom Navigation Bar)

```text
 ┌───────────────────────────────────────────────────────────┐
 │                                                           │
 │                        [콘텐츠 영역]                       │
 │                                                           │
 ├───────────────────────────────────────────────────────────┤
 │  [홈]  │  [검색/추천]  │  [커뮤니티]  │  [알람]  │  [배경화면] │
 └───────────────────────────────────────────────────────────┘
Tab 1. 홈 (Home)
미니멀 레이아웃: 첫 화면은 복잡한 요소 없이 '오늘의 명언' 카드와 공유/북마크 버튼만 배치하여 메시지에 집중하게 만듭니다.
퀵 테마 스위처: 우측 상단 혹은 햄버거 메뉴를 통해 터치 한 번으로 앱의 테마 색상을 변경할 수 있습니다.
Tab 2. 검색 및 감정 맞춤 추천 (Search & Recommend)
단순화된 통합 검색: 검색어 입력 시 실시간 인물/키워드 매칭을 지원합니다.
마음 분석기: 상단의 감정 입력 창에 사용자가 현재 상황이나 감정("지치고 힘들어", "중요한 면접이 있어")을 적으면, 자연어 분석을 통해 위로와 용기를 주는 맞춤형 명언 카드를 추천합니다.
Tab 3. 커뮤니티 (Community)
하트(좋아요) 소통: 유저들이 자신만의 동기부여 문장이나 목표를 공유하고, 서로 하트를 누르며 긍정적인 영향력을 주고받습니다.
월간 명언 인기 차트: 한 달 동안 유저들에게 가장 많은 하트를 받은 인기 명언과 유저 한마디를 차트 형식으로 직관적으로 노출합니다.
Tab 4. 알람 (Alarm)
모바일 전용 알람 컨트롤: 엄지손가락으로 조작하기 쉬운 큰 다이얼이나 휠 형태의 시간 설정 인터페이스를 채택합니다.
미션형 해제: 잠을 깨기 위해 정해진 명언 문구를 타이핑하여 알람을 끄는 입력 패드를 중앙에 넓게 배치하여 터치 오작동을 최소화합니다.
Tab 5. 배경화면 제작 (Wallpaper Generator)
한 손 편집기: 한 화면 내에서 명언 선택, 폰트 조절, 배경 이미지 지정을 모바일 하단 슬라이더 인터페이스로 간편하게 조작할 수 있도록 디자인합니다.
원터치 저장: 별도의 복잡한 과정 없이 하단 '저장' 버튼을 누르면 기기 내부 갤러리에 해상도 맞춤형 고화질 이미지로 저장됩니다.

5. 사용자 시나리오 (User Flow)
[앱 최초 실행]
   │
   ├──▶ [기본 테마 설정] (사용자 맞춤 컬러 선택)
   │
   └──▶ [메인 화면 진입]
          │
          ├──▶ [하단 Tab 1] 홈 ──▶ 오늘의 명언 감상 (공유 / 테마 색상 즉시 변경)
          │
          ├──▶ [하단 Tab 2] 검색/추천 ──▶ 검색 또는 감정 상태 입력 ──▶ 맞춤 명언 제안
          │
          ├──▶ [하단 Tab 3] 커뮤니티 ──▶ 다른 유저 글 조회 ──▶ 하트 누르기 및 월간 인기차트 확인
          │
          ├──▶ [하단 Tab 4] 알람 ──▶ 터치 친화적 시간 설정 ──▶ 기상 시 TTS 및 타이핑 해제 미션
          │
          └──▶ [하단 Tab 5] 배경화면 ──▶ 슬라이더 조작 스타일 편집 ──▶ 기기에 즉시 저장

6. 향후 확장 계획 (Future Roadmap)
위젯(Widget) 지원: 앱을 켜지 않고도 모바일 배경화면에서 오늘의 명언을 바로 볼 수 있는 기기별 시스템 위젯 개발.
소셜 로그인 연동: 커뮤니티 편의성을 증대하기 위한 간편 소셜 로그인(카카오, 애플, 구글) 기능 탑재.
