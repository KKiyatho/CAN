### [2026-06-13]

**AI와 대화했던 내용**
- Firebase Firestore 대용량 데이터(36,937개)를 무료 플랜(Spark) 한도 내에서 안전하게 삽입하기 위한 배치 실행 스크립트(`seed_all.js`) 작성 전략 논의.
- 앱 내에서 한국어 명언만 필터링하여 보여주는 기능 및 검색 탭의 UI를 Apple Music 스타일로 고도화하는 방법 탐색.
- 카테고리 클릭 시 상세 화면으로 이동하며, 무한 스크롤(Pagination)과 로컬 캐싱을 결합하여 성능과 비용을 최적화하는 아키텍처 설계.

**AI와의 대화를 통해 새롭게 알게 된 점**
- Firestore의 `startAfterDocument`를 활용한 커서 기반 페이지네이션이 `offset` 방식보다 대용량 데이터 처리에 효율적이고 인덱스 구성이 단순하다는 점을 배움.
- Flutter에서 `Container`의 그라디언트와 `Image.network`를 `Stack`으로 조합하고 `Color.withValues(alpha: ...)`를 사용해 Apple Music과 같은 고급스러운 블렌딩 효과를 내는 UI 기법을 익힘.
- `SharedPreferences`를 활용해 첫 페이지 데이터를 캐싱함으로써 사용자 경험을 개선하고 API 호출 비용을 절감하는 패턴을 적용함.

**이번 주차에 본인이 진행한 활동**
- `seed_all.js` 작성: 전체 명언 데이터를 499개 단위 배치로 분할 삽입하고 지연 시간을 두어 Firebase 할당량 초과 방지.
- `quote_repository.dart` 수정: `language == 'ko'` 조건 추가 및 페이지네이션 로직 구현.
- `SearchScreen` 전면 재설계: Unsplash 이미지를 활용한 12개 직업/주제별 카테고리 카드 UI 구현.
- `CategoryDetailScreen` 구현: 상단 유동 헤더(SliverAppBar) 및 무한 스크롤 리스트 뷰 완성.

**어려웠던 점이나 다음에 더 해보고 싶은 것**
- 초기에 `Clip.none` 설정으로 인해 UI가 깨지는 문제가 있었으나 `ClipRRect`와 `overflow: hidden` 구조로 수정하여 해결함.
- 다음에는 현재 하드코딩된 'ko' 언어 필터를 사용자 설정 탭에서 직접 변경할 수 있도록 언어 선택 기능을 추가해보고 싶음.
- 명언에 '좋아요' 기능을 추가하여 사용자별 맞춤 추천 시스템의 기초를 다지고 싶음.

### [2026-06-20]

**AI와 대화했던 내용**
- 분석을 통해 MVP 미완성 기능 4가지를 식별하고 전부 자동 구현 (Autopilot).
- 검색탭 감정 입력창, 배경화면 이미지 선택, 알람 휠 UI, 홈 좋아요 버튼, 언어 선택 기능 구현 논의.

**AI와의 대화를 통해 새롭게 알게 된 점**
- `ListWheelScrollView.useDelegate`와 `FixedExtentScrollController`로 iOS 스타일 피커와 같은 휠 UI를 Flutter에서 직접 구현 가능.
- Dart Record 타입 (`(String label, String url)`)을 활용해 간결한 이미지 목록 관리 가능.
- `ThemeState`에 `languageCode`를 추가해 앱 전역 언어 필터를 Riverpod 상태 관리로 처리하는 패턴.

**이번 주차에 본인이 진행한 활동**
- `theme_notifier.dart` 수정: `languageCode` 상태 추가, `setLanguage()` 메서드, `kLanguageOptions` 상수 추가.
- `quote_repository.dart` 수정: 모든 fetch 메서드에 `language` 파라미터 추가, `LikeRepository` 클래스 신규 추가.
- `home_notifier.dart` 수정: `likedIds` 상태 추가, `toggleLike()` 메서드 구현, 언어 필터 적용.
- `home_screen.dart` 수정: 좋아요 버튼 추가, 언어 선택 바텀시트 추가.
- `search_screen.dart` 수정: 감정/상황 전용 입력창 + 빠른 감정 칩 6개 추가, 검색 결과 헤더에 감정 문장 표시.
- `search_notifier.dart`, `category_detail_notifier.dart` 수정: 모든 검색 메서드에 언어 파라미터 전달.
- `wallpaper_screen.dart` 수정: 배경 이미지 6개 추가, 색상/이미지 탭 UI, `_BgTabButton`·`_BgImagePicker` 위젯 신규 추가.
- `alarm_screen.dart` 수정: `showTimePicker` 대신 `ListWheelScrollView` 기반 인라인 휠 UI로 교체.

**어려웠던 점이나 다음에 더 해보고 싶은 것**

- 다음에는 커뮤니티 탭에 게시글별 명언 '좋아요' 수를 Firestore에 집계하는 기능 추가해보고 싶음.
- 검색 탭의 감정 태그 매칭을 pgvector 기반 벡터 유사도 검색으로 고도화 시도 예정.
